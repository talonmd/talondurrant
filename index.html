Avarice Sheets

A web application running on React that utilizes a Node/Express server and MongoDB databse. This app allows users to register, sign in, and create their own character and campaign sheets for Abberations in Avarice, a role-playing game similar to Dungeons and Dragons. These character and campaign sheets are interactive and allow the user to perfrom all CRUD actions on the database. It makes use of NPM Packages such as Axios and Immer, and is built using React hooks and features such as Reducer and Context.

More features are continuously being built and added to complete this large project. In the future, features will be implemented to open live socket connections to allow instant messaging and sessions to be in use.

Note: I am currently working on the login feature, so some functionality is down on the home page. To access this App to see its features in work, go to the Login form and type "IMHUNGRY" all in lowercase into the email field, then press submit.

Botanicals, Inc.

A custom Wordpress theme created for Botanicals, Inc, a Utah-based Plantscape Design business. It was designed from scratch, and styled using Sass. This theme makes use of custom post types as well as custom wordpress queries, and was built with SEO in mind. In addition to a blog, the site has post types for Announcements, Events, and News to keep their retail locations happenings well known. The dev environment utilized Local by Flywheel as well as Node and NPM.
