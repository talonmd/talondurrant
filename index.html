Avarice Sheets

An in depth web application running on React that utilizes a Node/Express server and MongoDB databse. This app allows users to register, sign in, and create their own character and campaign sheets for Abberations in Avarice, a role-playing game similar to Dungeons and Dragons. These character and campaign sheets are interactive and allow the user to perfrom all CRUD actions on the database. It makes / will make use of NPM Packages such as Axios and Immer. They are currently under construction, and more features are being built and added constantly. Features will also be implemented to open live socket connections to allow instant messaging and sessions to be in use.

To access this App, go to the Login form and type "IMHUNGRY" all in lowercase into the email field, then press submit.

Botanicals, Inc.

A custom Wordpress theme created for Botanicals, Inc, a Utah-based Plantscape Design business. It was designed from scratch, and styled using Sass. This theme makes use of custom post types as well as custom wordpress queries, and was built with SEO in mind. In addition to a blog, the site has post types for Announcements, Events, and News to keep their retail locations happenings well known. The dev environment utilized Local by Flywheel as well as Node and NPM.
